作品集管理工具

新增、編輯作品後，按「儲存到專案」直接更新網站資料

自動儲存 — 資料會存在瀏覽器裡，關掉再打開不會消失

如何更新網站？

  1. 在這裡新增或編輯作品
  2. 按上方「儲存到專案」，選擇 portfolio 資料夾裡的 public/portfolio/data.json
  3. 回到終端機，執行 npm run deploy 即可上線

第一次使用時瀏覽器會請你選檔案位置，之後都會直接覆蓋同一個檔案。
